# Анализ вакансий для Data Scientist и рекомендательная система

## Описание проекта
Проект выполнен для кадрового агентства с целью создания модели машинного обучения для рекомендации вакансий соискателям на позицию Data Scientist. В рамках проекта проведен глубокий анализ рынка вакансий, требований работодателей и построена рекомендательная система.

## Цели проекта
1. Исследовать рынок вакансий для Data Scientist
2. Выявить ключевые требования работодателей
3. Проанализировать зарплатные ожидания по уровням опыта
4. Построить модель для рекомендации подходящих вакансий

## Источники данных
- **База данных:** PostgreSQL (project_sql)
- **Таблицы:** vacancies, areas, employers, industries, employers_industries
- **Объем данных:** 49,197 вакансий, 23,501 работодатель, 1,362 региона
